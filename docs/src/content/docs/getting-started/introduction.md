Tome is a self-hosted ebook library manager built with Rust and React.

## Quick Start

```bash
docker pull ghcr.io/unkos-dev/tome:latest
```

```bash
docker run -p 3000:3000 ghcr.io/unkos-dev/tome:latest
```

> **Note:** Tome is in pre-alpha. These instructions will be expanded as the project matures.
